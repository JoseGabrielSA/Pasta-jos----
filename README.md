<h1>Projeto Site Biostudy</h1>
-> 🌱 BioStudy

Projeto desenvolvido em equipe durante o curso técnico de Desenvolvimento de Sistemas.

## 📌 Sobre o projeto
O BioStudy é um site educacional focado na disciplina de Biologia, com o objetivo de compartilhar conhecimentos sobre a vida, os seres vivos e seus processos de forma clara, acessível e interessante.

## 🎯 Objetivo
Facilitar o aprendizado de Biologia para estudantes, utilizando uma abordagem simples, visual e organizada.

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 👥 Equipe
Projeto desenvolvido em grupo como atividade acadêmica.
